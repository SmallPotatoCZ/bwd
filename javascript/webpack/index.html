Webpack 构建

graph LR webpack[Webpack] --> entry[入口] click entry callNotes "entry" webpack[Webpack] --> output[输出] click output callNotes "output"

应该使用的模块，作为构建内部依赖图的开始。默认 ./src/index.js。

一个或多个入口：

{
    /*...*/
    entry:{
        entry1: 'file1.js',
        entry2: 'file2.js'
    },
    /*...*/
}

{
    /*...*/
    output:{
        filename: [name].bundle.js, // [name] 入口起点名字
        path: path.resolve(__dirname) // 输出路径
    }
}

资源管理 - loader

CSS

  1. style-css

  2. css-loader

    {
/*...*/ 
module:{
   rules:[
       {
           test: /\.css$/,
           use: ['style-loader', 'css-loader']
       }
   ]
}
/*...*/
}

图片

file-loader

{
   /*...*/ 
   module:{
       rules:[
           {
               test: /\.(png|svg|jpg|gif)$/,
               use: ['file-loader']
           }
       ]
   }
   /*...*/
}

字体

{
   /*...*/ 
   module:{
       rules:[
           {
               test: /\.(woff|woff2|eot|ttf|otf)$/,
               use: ['file-loader']
           }
       ]
   }
   /*...*/
}

数据

  1. csv-loader - TSV 和 CSV 文件
  2. xml-loader - XML 文件

导入的这些文件可以转换为 JSON 格式的数据。

常用插件

  1. webpack-html-plugin - 解决index.html文件中引用 bundle 的问题
  2. clean-webpack-plugin - 清空 dist 目录中文件
  3. BabelMinifyWebpackPlugin - 压缩 JavaScript
  4. ClosureCompilerPlugin - 压缩 JavaScript
  5. SplitChunksPlugin - 消除模块之间的重复引用
