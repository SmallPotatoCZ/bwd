文件 I/O

打开文件

目标 命令 备注
打开新的文件 :e <file> 支持通配符，如：**/<file>
垂直方向新建窗口并打开新建的文件 :new
水平方向新建窗口并打开新建的文件 :vnem
在当前窗口打开新的文件 :enew
在新的标签窗口打开新的文件 :tabnew
读文件内容 :r[ead] <file> 在锚点处添加<file>的内容

保存文件

目标 命令 备注
保存当前文件 :w
保存所有文件 :wa
当前文件另存为 :saves <file-name>

暂存文件操作

目标 命令
暂存文件列表 :ls
关闭当前缓存 :bd[elete]
跳转下一个标签页 :bn[ext]
跳转上一个标签页 :bp[revious]

水平方向切分窗口 :vsplit
窗口间移动锚点 ^w + <direction key>

标签页

目标 命令 备注
创建新的空标签页 :tabn[ew]
在新的标签页打开文件 :tabe[dit] <file or path>
切换标签页 :tabm[ove] <numeric value> 如：:tabm-1, :tabm+3（相对），:tabm 2（绝对）

文本编辑

目标 命令
编辑模式 i
在下一行新建一行并开始编辑 o
跳转到行的开始并开始编辑 I
跳转到行的结尾并开始编辑 A

缩进，制表符和空格

目标 命令 备注
开启空格格式的制表符 :set expandtab
设置制表符空格的数量 :set tabstop=<x> 一个制表符由x个空格组成。 softtabstop开启它不会生效
设置退格键删除的空格数量 :set softtabstop=<x> 一个退格键由x个空格组成。
设置制表符空格的数量 :set shiftwidth=<x> 每一级缩进的x个空格。只有expandtab开启，才会生效。
开启真正的制表符 :set noexpantab 返回默认的制表符格式

目标 命令 备注
删除行 dd
清除锚点到行尾的内容 D
清除整行内容 0D 0 移动锚点到行开始
清除整行并以插入模式结束 S

目标 命令 备注
字选择 v
块选择 ^ 字选择模式下，^ + <direction key>
行选择 V

选择括号内的内容 vi[parenthesis] 如：vi(、vi[和vi{
选择括号内的内容 vi[parenthesis] 如：vi(、vi[和vi{
选择括号内的内容（花括号） vib 如上：vi{

移动到行开始 0

文本的显示模式

目标 命令 备注
单词换行 :set wrap
显示换行符 :set linebreak
高亮当前行 :set cursorline
显示垂直行 :set colorcolumn=<x> x 是列号码
显示所有的空白 :set list
自动换行 :set textwidth=<x> 文本在您键入时自动换行，以使每行最多长 <x> 个字符
